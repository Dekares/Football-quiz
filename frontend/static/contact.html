03

İyi bir maç raporu nasıl yazılır?

  1. Günlük Tahmin mi, Solo mu kullandığını belirt.
  2. Oyuncu veya kulüp adı görünüyorsa aynen yaz.
  3. Beklediğin sonuç ile gördüğün sonucu ayrı ayrı anlat.
  4. Mümkünse ekran görüntüsü, tarih, cihaz ve tarayıcı ekle.

Bu bilgiler sorunu tekrar üretmemizi ve doğru kaydı hızlıca bulmamızı kolaylaştırır. Veri düzeltmeleri kaynaklarla karşılaştırılır ve doğrulanırsa sonraki veritabanı yayınında oyuna alınır.

02

What can you report?

Game bug

If a page fails, a guess cannot be submitted, a result is wrong or mobile layout breaks, include your device and browser.

Football data

Share the player, the missing or incorrect club, the right date and a reliable source when available.

Product idea

For a new mode, balance change or interface idea, briefly explain the player problem it would solve.

Collaboration

For content, advertising or project proposals, include the scope, objective and expected schedule.

03

How to write a useful match report

  1. Say whether you were in Daily Guess or Solo.
  2. Include the exact player or club name shown.
  3. Separate what you expected from what actually happened.
  4. Add a screenshot, date, device and browser when possible.

These details help us reproduce the problem and locate the correct record. Data corrections are compared with source material and, when verified, enter a later database release.

Useful first stops

  • How does the game work?
  • How is data prepared?
  • How is message data handled?
Security note

Do not include passwords, payment details, identity documents or other sensitive personal information.
